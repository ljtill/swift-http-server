Security Testing

Try these URLs to test the server's security features:

  • Parent Directory - Should be blocked/sanitized
  • Path Traversal - Should be blocked
  • Absolute Path - Should be contained to document root

Note: All dangerous paths are sanitized for security.

Server Information

  • Framework: SwiftNIO
  • Language: Swift 6.2+
  • Request Method: GET only
  • Document Root: Examples/Site/
