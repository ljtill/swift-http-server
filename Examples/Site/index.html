Welcome to Swift HTTP Server

A modern, lightweight HTTP server built with Swift and SwiftNIO for serving static files during local development.

Quick Start

Get up and running in seconds:

  • swift run HttpServer . - Serve current directory
  • swift run HttpServer ./Examples/Site --port 3000 - Custom directory and port
  • swift run HttpServer . --host 0.0.0.0 --port 8080 - Bind to all interfaces

Key Features

  • Static File Serving - Efficiently serve HTML, CSS, JavaScript, images, and more
  • MIME Type Detection - Automatic content-type headers for web files
  • Directory Listing - Browse directories when no index.html exists
  • Security Protection - Built-in directory traversal protection
  • Simple Logging - Console logging with timestamps for development
  • Graceful Shutdown - Clean server shutdown on SIGINT

Built with Swift 6

This server demonstrates modern Swift development practices:

  • SwiftNIO - High-performance, non-blocking I/O
  • Structured Concurrency - async/await for clean asynchronous code
  • Modular Architecture - Reusable HttpServerCore library
  • Comprehensive Testing - 9 tests across unit, integration, and end-to-end scenarios
